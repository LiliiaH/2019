== Installation ==

1. Add your social profile URLs and set custom icons (optional) under Settings > SimpleSocial.
1. Simply use the `[social]` shortcode on any page, post, or text widget or in your theme code, use `<?php echo do_shortcode( '[social]' ); ?>`.
